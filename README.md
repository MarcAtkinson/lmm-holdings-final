## Deploying the update
1. Unzip this file.
2. Copy all contents into the local `lmm-holdings` GitHub repository folder, replacing matching files.
3. Open GitHub Desktop.
4. Confirm the changed files.
5. Commit with summary: `Launch LMM Holdings Version 3`
6. Click `Push origin`.
7. Vercel will deploy automatically.
